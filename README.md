# test-calendar

Simple calendar app

Go to app: (Test-Calendar)[https://steelwinds.github.io/test-calendar/]

## Getting Started

### Installing 

You need cd in project repo, and install depends

```
cd ./test-calendar

pnpm i
```

### Development 

For run project on dev-server you can run 

```
pnpm serve
```

### Build 

For build project for production you can run 

```
pnpm build
```
